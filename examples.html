Examples

Usage

$ ./sudokuloeser -h
Usage:
  ./sudokuloeser [options] <input_file>
Options
  -U         Unicode borders
  -h         This help
  -H         HTML-Output
             twice: print on stdout, even if outfile given
  -o <file>  Output-File
  -O <file>  Overlay for non-standard files
  -c         No colors
  -p         Plaintext
  -n         Dont solve, just print
  -s         silent
Output:
  green:     Given values
  yellow:      Values find by logic
  white:      Values find by backtracking
