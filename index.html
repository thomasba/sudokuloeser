About

sudokuloeser is a simple sudoku solver written in C. It is optimized for speed.

Features

With sudokuloeser you can easily solve sudokus. As output there are several possibilties:

  • Plaintext
  • Unicode borders
  • With ASCII-Borders
  • HTML

Have a look on the examples page.
